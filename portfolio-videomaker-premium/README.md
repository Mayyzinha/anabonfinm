# Portfólio Premium de Videomaker

Projeto completo em Next.js + React + TypeScript + Tailwind CSS + Framer Motion + Lucide Icons.

## Personalização rápida

Edite seus dados em:

```txt
data/site.ts
```

Troque o vídeo principal em:

```txt
public/videos/showreel.mp4
```

Troque os projetos, clientes, depoimentos e estatísticas em:

```txt
data/portfolio.ts
```

## Rodar localmente

```bash
npm install
npm run dev
```

Abra no navegador:

```txt
http://localhost:3000
```

## Build de produção

```bash
npm run build
npm run start
```

## Publicação na Vercel

1. Suba este projeto no GitHub.
2. Acesse https://vercel.com.
3. Clique em Add New Project.
4. Importe o repositório.
5. Clique em Deploy.

A Vercel detecta automaticamente Next.js.
